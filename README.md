# Leave Track System

## Overview
Leave Track System is a MERN Stack application for managing employee leave requests.

## Features
- User Authentication (JWT)
- Employee Dashboard
- Admin Dashboard
- Leave Requests Management
- PostgreSQL Database
- Docker Support
- Docker Compose
- Terraform Infrastructure
- GitHub Actions CI

## Technologies
- React
- Node.js
- Express
- PostgreSQL
- Sequelize
- Docker
- Terraform
- GitHub Actions
- AWS
